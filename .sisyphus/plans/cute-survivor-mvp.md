# Cute Survivor MVP 开发计划（精简版）

> **项目代号**: 可爱幸存者 (Cute Survivor)
> **版本**: v1.0 MVP 精简版
> **创建时间**: 2026-04-04

---

## TL;DR

> **快速摘要**: 开发一款萌系俯视角射击生存游戏，验证核心玩法循环。

> **交付物**: 
> - 可运行的 Web 俯视角射击游戏
> - 玩家移动、瞄准、射击
> - 敌人 AI 与掉落
> - 简化地图生成

> **任务数**: 8 个核心任务
> **预估工作量**: 2-3 周

---

## 核心循环

```
玩家移动 → 射击敌人 → 击杀掉落 → 收集资源 → 制造弹药 → 继续战斗
```

---

## 任务列表

### T1. 项目初始化 + 基础渲染

**工作内容**:
- 初始化 Vite + TypeScript + Three.js 项目
- 配置 Vite（optimizeDeps、别名）
- 创建俯视角 OrthographicCamera
- 创建基础场景（地面网格、灯光）
- 窗口 resize 处理
- 游戏主循环框架（requestAnimationFrame + delta time）

**验收标准**:
- [ ] `npm run dev` 启动成功
- [ ] 俯视角 3D 画面显示
- [ ] 窗口缩放正常

---

### T2. 玩家移动 + 瞄准系统

**工作内容**:
- 创建玩家实体（简单几何体）
- WASD 键盘输入处理
- 基于 deltaTime 的移动
- 玩家旋转指向鼠标方向
- 移动边界限制

**验收标准**:
- [ ] WASD 控制玩家移动
- [ ] 玩家始终朝向鼠标方向
- [ ] 帧率无关的移动速度

---

### T3. 射击系统

**工作内容**:
- 手枪武器数据结构
- 鼠标点击发射子弹
- 子弹沿射击方向移动
- 子弹生命周期（超时消失）
- 射击冷却（射速限制）
- 弹药数量管理

**验收标准**:
- [ ] 点击鼠标发射子弹
- [ ] 子弹朝鼠标方向飞行
- [ ] 射击有冷却时间
- [ ] 显示弹药数量

---

### T4. 敌人实体 + AI

**工作内容**:
- 敌人基类（HP、速度、伤害）
- 敌人渲染（2-3 种颜色区分）
- 敌人生成（地图随机位置）
- AI 状态机：巡逻 → 警戒 → 追击 → 战斗
- 玩家进入范围触发追击

**验收标准**:
- [ ] 敌人在场景中生成
- [ ] 敌人有简单 AI（看到玩家会追击）
- [ ] 敌人会攻击玩家

---

### T5. 伤害系统 + 死亡掉落

**工作内容**:
- 子弹与敌人碰撞检测
- 伤害计算
- 敌人受伤反馈（颜色闪烁）
- 敌人 HP 归零死亡
- 死亡掉落物品（金币、材料）
- 玩家受伤逻辑

**验收标准**:
- [ ] 子弹击中敌人造成伤害
- [ ] 敌人死亡后掉落物品
- [ ] 敌人攻击减少玩家 HP

---

### T6. 简化地图生成

**工作内容**:
- 确定性随机数生成器（种子系统）
- 简化地形（草地 + 岩石障碍）
- 地形碰撞（岩石不可通行）
- 资源点生成（草药、矿石）
- 玩家出生点

**验收标准**:
- [ ] 相同种子生成相同地图
- [ ] 岩石区域不可通行
- [ ] 资源点分布在地图上

---

### T7. 物品系统 + 自动拾取

**工作内容**:
- 物品实体（场景中渲染）
- 掉落物品显示
- 玩家靠近自动拾取
- 物品添加到内存背包
- 物品数据管理

**验收标准**:
- [ ] 敌人掉落物品显示在场景
- [ ] 靠近物品自动消失并被收集
- [ ] 收集后物品数量增加

---

### T8. 基础 UI

**工作内容**:
- 游戏 HUD（HP、弹药、金币）
- 暂停菜单（ESC 打开）
- 主菜单（标题 + 开始按钮）
- 游戏结束界面
- Tab 键打开背包（显示物品）

**验收标准**:
- [ ] HUD 显示 HP、弹药、金币
- [ ] ESC 暂停游戏
- [ ] 玩家死亡显示 Game Over
- [ ] Tab 打开背包界面

---

## 最终验证

- [ ] F1. 核心循环测试：移动→射击→击杀→掉落→拾取
- [ ] F2. 构建测试：`npm run build` 成功

---

## 禁止实现（P2 扩展）

- NPC 交互系统
- 守护区域模式
- 换肤系统
- 合成系统（初版只做弹药）
- 地图编辑器

---

## 技术约束

- 使用 Three.js（不用其他 3D 引擎）
- 使用 OrthographicCamera（俯视角）
- 轻量碰撞检测（圆形距离检测，不用物理引擎）
- LocalStorage 存档

---

## 成功标准

```bash
npm run dev   # 开发服务器运行
npm run build # 生产构建成功
```
